Программа предназначена для обрезки pay.log по указанным шаблонам.

Для запуска рекомендуется использовать Start.bat (без него отжирает память)
Необходимо открыть Start.bat с помощью любого доступного текстового редактора, 
и в строке: "cd C:\Program Files\Java\jdk1.8.0_221\bin" - прописать путь до JRE/JDK
Если не жалко оперативной памяти, можно установить большее цифровое значение в параметре -Xmx256m.

Использование простое.
1) Необходимо выбрать файл для обработки
2) Папку для сохранения результата(Не обязательно, по-умолчанию сохранит в папку из которой был выбран файл)
3) Шаблон по которому проводить обработку
4) И ввести запрос.(Guid/Регулярное выражение)
ВНИМАНИЕ!!! 
Поисковой запрос чувствителен к регистру!!!
Результат сохранится:	для шаблона GUIDa - "d06da315-0565-472c-88e6-068619e4cfe8.txt", - где "d06da315-0565-472c-88e6-068619e4cfe8" гуид из строки поиска
						для регулярного выражения - "REGEX.txt"
